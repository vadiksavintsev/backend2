Gitclone

склонировал с репозитория на учебный сервер

GET HTTP/1.0

С помощью запроса GET в протоколе HTTP 1.0 получили главную страницу сайта kubsu-dev.ru
По умолчанию сервер отвечает по протоколу HTTP 1.1

GET HTTP/1.1

С помощью запроса GET в протоколе HTTP 1.1 получили внутренную страницу u52989.kubsu-dev.ru

HEAD file.tar.gz

С помощью запроса HEAD узнал информацию о файле file.tar.gz
В параметре "Content-Length" содержится информация о размере файла

HEAD image.png

С помощью запроса HEAD узнал информацию о файле image.png
В параметре "Content-Type" содержитс информация о типе файла

Comment

Отправил комментарий с помощью запроса POST, в теле которого был указан comment= "мой комментарий"
В ответ был получен "мой комментарий", а также строка "Привет мир!"

Range

С помощью запроса GET c заголовком Range=0-99 были получены 1-ые 100 байт файла file.tar.gz

Code

С помощью запроса HEAD c заголовком Accept-Charset: UTF-8 определил кодировку ресурса index.php
